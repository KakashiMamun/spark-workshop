web UI

  web UI is the web interface of a singe Spark application

  Monitoring and inspecting Spark jobs and tasks

  Opens at http://localhost:4040 (by default)

  One big SparkListener

  Composed of tabs and pages (on the following slides)

©Jacek Laskowski 2017 / @jaceklaskowski / jacek@japila.pl

web UI / Spark Properties

  Enabled by default unless spark.ui.enabled is turned off

  Use spark.ui.port to set the port number

  Actual availability and URL dependent on the deployment environment (e.g. YARN, Mesos, Standalone, local, Kubernetes)

©Jacek Laskowski 2017 / @jaceklaskowski / jacek@japila.pl

web UI Tabs (1 of 2)

  Jobs shows status of Spark jobs

  Stages shows status of stages of all Spark jobs

  Storage shows RDD size and memory use

  Environment shows the environment

  Executors shows status of executors

  SQL shows status of batch and streaming queries
